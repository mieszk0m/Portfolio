# 🌐 Portfolio Web Development

## 📌 Opis
Ten folder zawiera różne projekty związane z tworzeniem stron internetowych, interakcją użytkownika i JavaScript.

---

## 📂 Zawartość
### **🔹 Główna strona internetowa**
- **`strona.html`** – Strona internetowa systemu USOS.
- **`styles.css`** – Stylowanie strony USOS w CSS.

### **🔹 Interakcje JavaScript**
- **`zadanie1.js`** – Obsługa logów, alertów i `document.write()`.
- **`zadanie1A.html`** – Testowanie skryptów JS w HTML.
- **`zadanie15.html`** – Importowanie JS przez `<script src>`.

### **🔹 Formularze i interakcja użytkownika**
- **`zadanie10.html`** – Formularz wejściowy z `prompt()`.
- **`zadanie12.html`** – Obliczenia w formularzu HTML.
- **`zadanie13.html`** – Funkcja zwrotna dla formularza HTML.

---

## 🔧 Uruchamianie kodu
1. Otwórz dowolny plik `.html` w przeglądarce.
2. Jeśli plik używa JS (`zadanie15.html`), upewnij się, że `zadanie1.js` jest w tym samym folderze.
