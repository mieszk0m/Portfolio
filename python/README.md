# 🐍 Portfolio Python

## 📌 Opis
Ten folder zawiera różne projekty w Pythonie, obejmujące analizę matematyczną, system ocen USOS, ataki brute-force oraz testowanie aplikacji.

---

## 📂 Zawartość
### **🔹 Analiza matematyczna**
- **`sys_op.py`** – Obliczanie miejsc zerowych funkcji matematycznej metodą numeryczną.

### **🔹 System ocen USOS**
- **`funkcje_usos.py`** – Obsługa bazy ocen w systemie USOS.
- **`usos2.py`** – Obsługa komend w systemie USOS.
- **`usos4.py`** – Pełny system zarządzania ocenami studentów.
- **`usos5.py`** – Obsługa wykładów i ćwiczeń w USOS.
- **`usos6.py`** – Wersja systemu USOS z dodatkowymi funkcjami zarządzania studentami.

### **🔹 Cyberbezpieczeństwo**
- **`zlamaniehasla.py`** – Atak brute-force na SHA-1 (sprawdza miliony hashy).

### **🔹 Testowanie i operacje na danych**
- **`test.py`** – Testy jednostkowe i funkcje do operacji na danych w USOS.
