# 🌐 Portfolio Web Development

## 📌 Opis
Ten folder zawiera projekty związane z aplikacjami internetowymi, backendem w Node.js oraz interakcją użytkownika za pomocą JavaScript.

---

## 📂 Zawartość
### **🔹 Backend (Node.js & Express)**
- **`app1.js`** – Serwer Express obsługujący formularze AJAX i Fetch API.
- **`app2.js`** – Rozszerzona wersja serwera Express obsługująca różne formaty odpowiedzi.
- **`index.js`** – Główny serwer backend obsługujący statyczne pliki i JSON API.

### **🔹 Frontend (HTML, CSS, JavaScript)**
- **`formularz.html`** – Formularz HTML do obsługi zapytań użytkownika.
- **`styles.css`** – Arkusz stylów CSS dla interfejsu aplikacji.
- **`script.js`** – Obsługa dynamicznych interakcji użytkownika poprzez konsolę i formularze.

### **🔹 Formularze i interakcja użytkownika**
- **`zadanie10.html`** – Formularz wejściowy z dynamiczną obsługą `prompt()`.
- **`zadanie12.html`** – Obliczenia w formularzu HTML.
- **`zadanie13.html`** – Funkcja zwrotna dla formularza HTML.

### **🔹 Konfiguracja projektu**
- **`package.json`** – Konfiguracja Node.js i zarządzanie zależnościami.
